Static Media

  • Django Static Media
  • add django.contrib.staticfiles to settings.INSTALLED_APPS
  • Define STATIC_URL in settings.py
  • Place the files in APP/static/APP/
  • In a template: <img src="{{ STATIC_URL }}images/hi.jpg" />
STATIC_URL = 'http://localhost:8000/static/'

In a form:

class Media:
   js = (
           'django-support-page/js/domready.js',
           'django-support-page/js/events.js',
           'django-support-page/js/browserdetect.js',
           'django-support-page/js/json2.js',
           'django-support-page/js/modernizr-1.6.min.js',
           'django-support-page/js/detect.js',)
